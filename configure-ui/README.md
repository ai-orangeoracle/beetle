# AI Job Market

React + TypeScript + Vite + MUI 项目。

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
npm run preview   # 预览生产构建
```

## 技术栈

- **React 19** + **TypeScript**
- **Vite 7** 构建
- **MUI (Material UI) 7**：`ThemeProvider`、`CssBaseline` 已在 `src/main.tsx` 中配置，主题可在该文件内修改
