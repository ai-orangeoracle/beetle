# 显示仪表板

[English](../en-us/display.md) | **中文**

显示模块通过 SPI 连接的 TFT 屏幕实时展示运行状态仪表板。程序化绘制的甲壳虫图标直观反映系统状态，配合通道健康指示、IP 地址和堆压力进度条 —— 全程无需图片资源或外部字体。

---

## 支持的显示控制器

| 控制器 | 典型分辨率 | 说明 |
|--------|-----------|------|
| **ST7789** | 240x240、240x320 | 默认选择；面板原生色彩反转 ON |
| **ILI9341** | 240x320 | 色彩反转默认 OFF |

两者使用相同的 SPI 初始化序列（SWRESET → SLPOUT → COLMOD → MADCTL → INV → NORON → DISPON）。`invert_colors` 配置项用于翻转驱动器的默认反转行为。

---

## 硬件接线

ESP32-S3 的典型 SPI 接线：

| 信号 | 说明 | 必需 |
|------|------|------|
| SCLK | SPI 时钟 | 是 |
| MOSI | SPI 数据输出（MISO 未使用） | 是 |
| CS | 片选 | 是 |
| DC | 数据/命令选择 | 是 |
| RST | 硬件复位（初始化时拉低→拉高） | 可选 |
| BL | 背光使能（初始化时拉高） | 可选 |

> 所有引脚号为 ESP32-S3 GPIO 编号。SPI host 必须为 2（HSPI）或 3（VSPI）。

---

## 配置

显示配置存储在 SPIFFS 分区的 `config/display.json` 中，也可通过配置 Web UI（显示 Display 页面）编辑。

### 完整配置示例

```json
{
  "version": 1,
  "enabled": true,
  "driver": "st7789",
  "bus": "spi",
  "width": 240,
  "height": 240,
  "rotation": 0,
  "color_order": "rgb",
  "invert_colors": false,
  "offset_x": 0,
  "offset_y": 0,
  "spi": {
    "host": 2,
    "sclk": 42,
    "mosi": 41,
    "cs": 21,
    "dc": 40,
    "rst": 39,
    "bl": 38,
    "freq_hz": 40000000
  }
}
```

### 字段说明

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `version` | u32 | 1 | 配置模式版本号，必须为 `1` |
| `enabled` | bool | — | 启用/禁用显示。`false` 时不初始化任何 SPI 硬件 |
| `driver` | string | — | `"st7789"` 或 `"ili9341"` |
| `bus` | string | — | `"spi"`（当前唯一选项） |
| `width` | u16 | — | 面板宽度（像素，1–480） |
| `height` | u16 | — | 面板高度（像素，1–480） |
| `rotation` | u16 | 0 | 显示旋转角度：`0`、`90`、`180` 或 `270` |
| `color_order` | string | `"rgb"` | `"rgb"` 或 `"bgr"` |
| `invert_colors` | bool | false | 翻转驱动器默认的色彩反转行为 |
| `offset_x` | i16 | 0 | 显示窗口水平偏移（-480 到 480） |
| `offset_y` | i16 | 0 | 显示窗口垂直偏移（-480 到 480） |
| `spi.host` | u8 | 2 | SPI 主机：`2`（HSPI）或 `3`（VSPI） |
| `spi.sclk` | i32 | — | SPI 时钟 GPIO 引脚 |
| `spi.mosi` | i32 | — | SPI MOSI GPIO 引脚 |
| `spi.cs` | i32 | — | 片选 GPIO 引脚 |
| `spi.dc` | i32 | — | 数据/命令 GPIO 引脚 |
| `spi.rst` | i32? | null | 复位 GPIO 引脚（可选） |
| `spi.bl` | i32? | null | 背光 GPIO 引脚（可选） |
| `spi.freq_hz` | u32 | 40000000 | SPI 时钟频率（1–80 MHz） |

## 仪表板布局

仪表板在纯白背景上渲染：

```
┌──────────────────────────────────┐ y=0
│ ┌──────┐                         │
│ │甲壳虫│  状态名称                │ 标题（粗体）
│ │64x64 │  192.168.1.100           │ IP 副标题
│ └──────┘                         │
├──────────────────────────────────┤ y≈104
│  ● telegram    ● feishu          │ 通道状态
│  ○ dingtalk    ● wecom           │ ● 实心=已启用
│  ● qq_channel                    │ ○ 空心=未启用
├──────────────────────────────────┤ y≈168
│  NORMAL   ████████░░░░ 62%       │ 压力等级 + 堆使用进度条
└──────────────────────────────────┘ y=240
```

### 甲壳虫图标状态

甲壳虫完全由 `embedded-graphics` 原语（圆、线）绘制，不使用任何位图或 PNG 资源。

| 状态 | 甲壳虫颜色 | 视觉特征 |
|------|-----------|----------|
| **启动中 (Booting)** | 橙色 | 身体上的加载圆点；头上方虚线 WiFi 弧线；副标题显示固件版本号 |
| **无 WiFi (NoWifi)** | 灰色 | 头上方实线 WiFi 弧线，被红色 X 划掉 |
| **空闲 (Idle)** | 绿色 | 身体上的白色对勾 ✓ |
| **繁忙 (Busy)** | 蓝色 | 鞘翅下展开的膜翅 |
| **异常 (Fault)** | 红色 | 翻转（头朝下）；X 眼；身体上的白色感叹号 |

### 通道状态指示

| 视觉样式 | 含义 |
|----------|------|
| 绿色实心圆 + 正常文字 | 已启用通道，健康 |
| 红色实心圆 + 正常文字 | 已启用通道，不健康 |
| 灰色空心圆 + 暗淡文字 | 未启用通道 |

### 底部栏

- 压力等级标签：**NORMAL**（绿色）/ **CAUTIOUS**（黄色）/ **CRITICAL**（红色）
- 堆使用率进度条 + 百分比数字

## 局部更新

为减少 SPI 流量，常变区域使用局部行刷新：

| 命令 | 更新内容 | 刷新行数 |
|------|---------|----------|
| `UpdateIp` | 仅 IP 地址副标题 | ~16 行 |
| `UpdatePressure` | 仅底部压力进度条 | ~72 行 |
| `RefreshDashboard` | 全屏 | 全部行 |

局部行刷新相比全屏刷新可减少约 85% 的 SPI 数据传输量。

## 内存影响

| 项目 | 大小 | 位置 |
|------|------|------|
| 帧缓冲 (240x240 RGB565) | 115,200 字节 | PSRAM |
| `SpiDisplayBackend` 结构体 | ~64 字节 | 堆（internal） |
| 渲染栈使用 | ~200 字节 | 栈 |
| **净 internal DRAM 影响** | **约 0** | PSRAM 承担帧缓冲开销 |

显示线程使用 6 KB 栈，每 5 秒从 `orchestrator::snapshot()` 读取快照并刷新。

## 注意事项与技巧

1. **`enabled: false` 表示零硬件初始化** — 不初始化 SPI 总线、不配置 GPIO、不分配 PSRAM。即使没有连接显示屏，也可以安全地保留在配置中。

2. **偏移值 (offset)** — 部分面板（尤其是 240x240 ST7789 使用 240x320 玻璃基板时）需要设置 `offset_x` 或 `offset_y` 来移动可视窗口。常见场景：240x240 面板在 240x320 基板上时设置 `offset_y: 80`。

3. **色彩顺序 (color_order)** — 如果颜色出现红蓝互换，尝试在 `"rgb"` 和 `"bgr"` 之间切换。

4. **色彩反转 (invert_colors)** — 如果显示效果像照片负片，切换此标志。ST7789 面板通常需要反转 ON（默认行为），ILI9341 面板通常需要反转 OFF。

5. **SPI 频率** — 40 MHz 适用于大多数面板。如果出现花屏或显示异常，可尝试降低到 20 MHz 或 10 MHz。最高支持 80 MHz。

6. **显示线程栈** — 渲染线程使用 6 KB 栈，对当前仪表板布局已足够。如果增加显著复杂的渲染逻辑，需注意栈溢出风险。

7. **Host 编译** — 在非 ESP 目标上（`cargo check`、`cargo clippy`），显示后端返回 `available: false`，所有命令为空操作。确保代码库在 host 端干净编译。

8. **旋转 (rotation)** — `rotation` 字段在控制器层面应用 MADCTL 变换。帧缓冲尺寸（`width` x `height`）应匹配旋转后的可视区域。
